# Flexgrid

Flexgrid is a lightweight, responsive grid Layout using Flexbox.

## Usage
Example
````
<div class="row">
  <div class="col-4-12"><h2>4/12</h2></div>
  <div class="col-8-12"><h2>8/12</h2></div>
</div>
````

## Available widths
- col-1-12
- col-2-12
- col-3-12
- col-4-12
- col-5-12
- col-6-12
- col-7-12
- col-8-12
- col-9-12
- col-10-12
- col-11-12

## Media widths
The columns will stack when the width is less than 600 px.
